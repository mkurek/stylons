Action Class¶

class stitems.Action(action=None, url=None, dataID=None, componentID=None)¶

Bases: dict

Action component Defines actions performed i.e. after click on the button, click on the list and so on

Keyword arguments:

action (string)
action to be executed, i.e. load, send, popupShow, popupHide, clear
url (string)
url to load/send data
dataID (string)
ID of compontent, from which data will be sended to server
componentID (string)
id of component to clear
